Sobre a aplicaçãoObjetivo:

O objetivo da aplicação é testar a interação da Web Speech API com animações CSS. Não há um objetivo do jogo em si.

Tecnologia Assistiva/Necessidade especial:

A Web Speech API pode ser considerada uma ferramenta para pessoas com dificuldade para utilizar mouse, teclado, e outros dispositivos que exijam habilidade motora. A página também busca atender requerimentos de acessibilidade padrão em HTML.
